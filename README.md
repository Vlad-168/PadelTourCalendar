# Padel Tour Calendar

Мобильный PWA-календарь турниров **FIP / Premier Padel** с удобными фильтрами —
быстрая альтернатива громоздкому календарю на [padelfip.com](https://www.padelfip.com/calendar/).

## Возможности

- **Календарь по месяцам** — турниры отмечены цветными точками по категории,
  тап по дню открывает список турниров этого дня.
- **Список турниров**, сгруппированный по месяцам.
- **Фильтры**: поиск по названию/городу/стране, категория турнира (Premier
  Padel Major/P1/P2, FIP Platinum/Gold/Silver/Bronze и др.), статус (идут
  сейчас / скоро / завершены), страна.
- **Карточка турнира**: даты, город, страна, кнопка «Добавить в календарь»
  (скачивает `.ics`) и ссылка на официальный календарь FIP.
- Устанавливается на телефон как приложение (PWA, работает офлайн — данные
  турниров зашиты в сборку).

## Данные

Турниры взяты из официального календаря FIP и покрывают период
**8 января – 19 июля 2026**. Список — в `src/data/tournaments.ts`, формат
каждой строки: `[категория, название, город, страна, дата начала, дата
окончания]`. Чтобы обновить или расширить календарь, допишите новые строки в
этот файл (или замените источник данных на живой API/скрапер).

## Разработка

```bash
npm install
npm run dev
```

Приложение полностью статическое (без бэкенда и авторизации) — все данные
подключаются во время сборки.

## Сборка

```bash
npm run build
npm run preview
```

## Деплой

Это обычное статическое SPA (Vite build → `dist/`), можно задеплоить куда
угодно: Vercel, Netlify, Cloudflare Pages, GitHub Pages, Firebase Hosting.
Например, для Vercel/Netlify достаточно указать команду сборки `npm run
build` и папку `dist` как output.
